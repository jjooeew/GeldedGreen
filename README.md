# Gelded Green

A small Vite/React prototype for a plant business landing page and category shop view.

## Current Scope

- Landing page with five product categories.
- Shop/category page with product list, selected product image, description, price, and an add-to-bag button.
- Products, copy, and images are placeholders for visual review.

Cart, enquiry, and checkout functionality are intentionally out of scope until the visual design is approved.

## Local Development

Prerequisite: Node.js 20 or newer.

```bash
npm install
npm run dev
```

## Production Check

```bash
npm run lint
npm run build
```

## Vercel

Vercel can deploy this as a standard Vite project:

- Build command: `npm run build`
- Output directory: `dist`
